# 💬 ChatBot

A smart and interactive chatbot built using **Python (Flask)**, **HTML**, **CSS**, and **JavaScript**. This chatbot can process user input, correct spelling errors, identify **prefixes/suffixes**, and perform **part-of-speech (POS) tagging** using Natural Language Processing techniques.

---

## 🚀 Features

- 🔍 **Spell Correction** using `pyspellchecker`
- 🧠 **Part-of-Speech (POS) Tagging** using `spaCy`
- 🧩 **Prefix & Suffix Detection** to identify root words
- 🗣️ Optional **Speech Input/Output** (Web Speech API)
- 💬 Clean chat UI using HTML, CSS (Responsive)
- ⚙️ Easy to run on any system with Python

---

## 🛠️ Tech Stack

- **Backend**: Python, Flask
- **Frontend**: HTML, CSS, JavaScript
- **NLP Libraries**: spaCy, pyspellchecker
